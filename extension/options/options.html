These extensions are chrome devtools extensions with two differences:

  1. HTML files must include these .js files: 
    <!-- These files are equivalent to devtools_extension_api.js -->
<script src='chrome-extension://ggimboaoffjaeoblofehalflljohnfbl/WebInspectorKit/Source/WebCore/inspector/front-end/ExtensionAPI.js'></script>
<script src='chrome-extension://ggimboaoffjaeoblofehalflljohnfbl/WebInspectorKit/Source/WebKit/chromium/src/js/DevToolsExtensionAPI.js'></script>
<script src='chrome-extension://ggimboaoffjaeoblofehalflljohnfbl/atopwi/buildExtensionAPI.js'></script>
<!-- RPC Wrapper for chrome.debugger.sendCommand -->
<script src="chrome-extension://ggimboaoffjaeoblofehalflljohnfbl/atopwi/DebuggerProtocol.js" ></script>
    These files mimic code injected into devtools extension .html files.
  2. More features are available:
    • remote debug protocol
    • ExtensionItemSelector
Load these extensions as normal Chrome devtools extensions and put the URL chrome-extension://id/path-to-html in the table above.
